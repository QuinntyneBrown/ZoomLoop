# Customer Support - Frontend Specification

**Version:** 1.0
**Date:** December 23, 2025
**Feature:** Customer Service & Support

---

## Overview

The Customer Support feature enables customers to get help, submit inquiries, track support tickets, and provide feedback.

---

## User Stories

### US-1: Contact Support
**As a** customer
**I want to** contact customer support
**So that** I can get help with my issue

### US-2: Track Support Tickets
**As a** customer
**I want to** track my support requests
**So that** I know the status

### US-3: Provide Feedback
**As a** customer
**I want to** share my experience
**So that** I can help improve the service

## Page Routes

| Route | Component | Description |
|-------|-----------|-------------|
| `/help` | HelpCenterPage | Help landing |
| `/help/articles/:id` | ArticlePage | Help article |
| `/help/contact` | ContactPage | Contact options |
| `/help/tickets` | TicketsListPage | My tickets |
| `/help/tickets/new` | NewTicketPage | Submit ticket |
| `/help/tickets/:id` | TicketDetailPage | Ticket view |
| `/feedback/:purchaseId` | FeedbackPage | Post-purchase survey |

---

## API Integration

```
GET /api/v1/help/articles
  Query: q, category
  Response: { articles: Article[] }

POST /api/v1/support/tickets
  Body: { topic, vehicleId, subject, message, attachments }
  Response: { ticketId, ticketNumber }

GET /api/v1/support/tickets
  Response: { tickets: Ticket[] }

GET /api/v1/support/tickets/:ticketId
  Response: TicketDetails

POST /api/v1/support/tickets/:ticketId/reply
  Body: { message, attachments }

POST /api/v1/feedback
  Body: { purchaseId, ratings, nps, comments }

POST /api/v1/support/callback
  Body: { phone, preferredTime }
```
