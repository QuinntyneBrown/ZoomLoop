# Search Bar Component

## Overview
The search bar is a prominent input component that allows users to search for vehicles. It can include filters, suggestions, and quick actions.

## Component States

| State | Description |
|-------|-------------|
| Default | Empty search bar ready for input |
| Focused | Active with cursor |
| With Query | Contains search text |
| With Suggestions | Dropdown showing suggestions |
| Loading | Searching in progress |
| No Results | No matches found |
| With Filters | Advanced filters visible |

## Keyboard Navigation

| Key | Action |
|-----|--------|
| Enter | Submit search or select highlighted suggestion |
| Escape | Close suggestions, clear search |
| Arrow Down | Move to next suggestion |
| Arrow Up | Move to previous suggestion |
| Tab | Move to next interactive element |
