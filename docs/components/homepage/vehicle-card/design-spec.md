# Vehicle Card Component

## Overview
The vehicle card is the primary representation of a vehicle listing. It displays key vehicle information in a scannable, clickable format used throughout the platform in grids and carousels.

## Component States

| State | Description |
|-------|-------------|
| Default | Standard card with image and details |
| Hover | Elevated with subtle shadow increase |
| Loading | Skeleton placeholder |
| Favorited | Heart icon filled |
| Sold/Reserved | Overlay indicating status |
| Featured | Highlighted with special styling |

### Hover Effects
- Card elevation increases (shadow)
- Image can slightly zoom (1.02x)
- Cursor: pointer

- Entire card is clickable
- Navigates to vehicle detail page
- Favorite button has separate click handler
